Miniclip AI Pipeline

Production-ready CLI pipeline для превращения длинного видео в готовый контент под несколько платформ.

Задача

Автоматизировать процесс: input video → audio extraction → transcription → генерация клипов → SEO-тексты → мультиплатформенный контент → delivery ZIP.

Моя роль

Полная архитектура, реализация, тестирование и документация.

Стек

Python CLI, Whisper, Ollama/qwen, FFmpeg, JSON manifest, platform renderers, smoke/release checks.

Ключевые решения

Многоэтапный пайплайн с промежуточными чекпоинтами, автоматическое packaging, локальная privacy-first обработка.

Результат

10+ клипов за один запуск, готовый ZIP для публикации, repeatable production процесс и экономия десятков часов ручной работы.

Learnings

Сразу думать о delivery и документации важнее, чем о красивом isolated demo.
