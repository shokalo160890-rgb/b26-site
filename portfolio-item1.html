Miniclip AI Pipeline

Production-ready pipeline for extracting video value into short clips, platform content and delivery packages.

Задача

Собрать single-command workflow: input video → audio extraction → transcription → clips → content units → platform texts → delivery ZIP.

Stack

Python CLI, Whisper, Ollama/qwen, FFmpeg, JSON manifest, platform renderers, smoke/release checks.

Результат

10 clips, delivery/manifest.json, quick_publish assets, 5 platform outputs and repeatable production packaging.
