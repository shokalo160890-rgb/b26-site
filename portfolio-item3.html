Case · Infrastructure

Local Dev-Cloud Infrastructure

Self-hosted мини-облако для быстрой разработки и демонстрации AI/web проектов.

Задача

Поднять удобную локальную инфраструктуру с reverse proxy, панелью управления, автоматизацией, базами данных и AI-сервисами.

Моя роль

Полная настройка Docker Compose, маршрутизация, документация и workflow для повторного запуска.

Стек

Docker Compose, Traefik, Portainer, n8n, Redis, Postgres, Ollama, Cloudflare Tunnel.

Ключевые решения

Один адрес для сервисов, быстрый deploy, шаблоны для новых проектов, локальный AI backend.

Результат

Быстрый self-hosted контур для экспериментов, демо и production-like тестирования.

Learnings

Хорошая dev-инфраструктура ускоряет все остальные проекты в разы.
