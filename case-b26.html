CASE STUDY

B26 Online — персональная цифровая инфраструктура

B26 Online — первый demo-case продукта: отдельный production-комплект под человека, услугу, портфолио или малый бизнес.

Открыть сайт Обсудить проект

Задача

Создать не просто сайт, а независимую цифровую инфраструктуру: собственный домен, сервер, HTTPS, публичный сайт, мобильная админка, управление контентом, backup, monitoring и полный контроль.

Решение

  • домен b26.online
  • VPS на Ubuntu
  • Nginx
  • HTTPS через Let's Encrypt
  • публичный сайт
  • GitHub deploy
  • JSON CMS
  • Flask admin panel
  • PWA mobile admin
  • Content Control Panel V2.1
  • Media Manager
  • upload / preview / copy URL / open / delete
  • backup
  • health-check
  • security: /.git blocked
  • final production snapshot

Результат

Система получила финальный статус READY / CLEAN / STABLE. B26 Online теперь используется как live-demo коммерческого продукта.

Формула продукта

Один человек = один домен + один сайт + одна мобильная админка + один backup-контур.
