# B26 Site

**Мини-сайты под смартфон**

Одна ссылка вместо кучи объяснений.

## Суть проекта

Каждый человек со смартфоном может иметь свой мини-сайт.

Мини-сайт работает как:

- визитка
- портфолио
- прайс
- контактная страница
- продавец
- QR-страница

## Главная ценность

Клиенту не нужно долго объяснять, кто он и что делает.

Он отправляет одну ссылку в WhatsApp, Telegram, Instagram или TikTok — и человек сразу видит:

- кто он
- что предлагает
- фото / примеры работ
- цены
- контакты
- кнопку связи
